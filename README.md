# HTML/CSS Giallo Booleano 👨‍🍳

Esercizio di replica e adattamento responsive di un layout editoriale dedicato al mondo del food. Il focus del progetto è la trasformazione di una pagina statica in un'esperienza cross-device completa, rispettando il vincolo di non alterare il codice sorgente HTML originale.

---

### 🎯 Obiettivo
Testare le conoscenze su:
* **Media Queries**: Gestione precisa dei breakpoint per adattare il layout a smartphone, tablet e desktop.
* **Non-Destructive CSS**: Capacità di sovrascrivere gli stili esistenti tramite un foglio di stile esterno, mantenendo intatto l'HTML di partenza.
* **Layout Adaptivity**: Trasformazione dei flussi da verticali (mobile) a griglie complesse (desktop).

### 🛠️ Tecnologie Utilizzate
* **HTML5**: Struttura semantica di base (fornita).
* **CSS3**: Flexbox, Media Queries e proprietà di posizionamento avanzate.
* **Mobile-First**: Metodologia di sviluppo che parte dalla risoluzione minima per aggiungere complessità salendo di risoluzione.
