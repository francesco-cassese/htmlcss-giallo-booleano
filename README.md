TML/CSS Giallo Booleano 👨‍🍳
Esercizio di replica e adattamento responsive di un layout editoriale dedicato al mondo del food. Il focus del progetto è la trasformazione di una pagina statica in un'esperienza cross-device completa, senza alterare il codice sorgente originale.

🎯 Obiettivo
Testare le conoscenze su:

Media Queries: Gestione dei breakpoint per adattare il layout a diverse risoluzioni.

Non-Destructive CSS: Sovrascrittura degli stili esistenti tramite un file CSS esterno senza modificare l'HTML originale.

Layout Adaptivity: Passaggio da flussi verticali (mobile) a strutture a griglia e affiancate (tablet/desktop).

🛠️ Tecnologie Utilizzate
HTML5: Struttura semantica fornita (non modificabile).

CSS3: Utilizzo di Flexbox e proprietà di posizionamento.

Mobile-First: Sviluppo ottimizzato partendo dalla base mobile fornita.
